Byrng is a Minecraft server maintained by Splendorr.

Currently Running Unmodded Minecraft 1.8.1, because Bukkit has proven unstable with the rapidity of the last few updates.

Read the changelog when you first open Minecraft to see what has changed in each new version, and let me know if you encounter weirdness.

In lieu of more sophisticated permissions control, I've set up a whitelist for the server. You cannot log in unless you are on the whitelist. To be added, please email simolinic+byrng@gmail.com with a request, including your relationship to any current members.
